Kitch n Break — Zusammenfassung der Implementierung

Erstellt:

Ziel

Ein kleines React-Projekt (Vite) mit Rezeptverwaltung — Home, Rezeptliste, Rezeptdetails, Formular zum Erstellen von Rezepten und Login. Integration mit Firebase (Auth, Firestore, Storage) und Hosting-Konfiguration vorbereitet.

Was wurde umgesetzt

  • Projekt-Scaffold mit Vite + React.
  • Kernseiten: Home, Rezeptdetails, CreateRecipe, Login.
  • Komponenten: Navbar, Footer, RecipeCard, RecipeList, SearchBar, InputField, Button.
  • Firebase-Integration: src/firebase.js exportiert auth, db, storage, analytics.
  • Firestore-Services: src/services/recipes.js (fetch/create).
  • Bilder-Upload in Firebase Storage aus dem Create-Formular.
  • Responsives Styling und Such-/Filter-Funktion in der Home-Ansicht.
  • Hosting-Konfiguration: firebase.json und .firebaserc (public: dist).

Wesentliche Dateien

  • index.html, src/main.jsx, src/App.jsx
  • Seiten: src/pages/Home.jsx, src/pages/RecipeDetails.jsx, src/pages/CreateRecipe.jsx, src/pages/Login.jsx
  • Komponenten: src/components/*
  • Firebase: src/firebase.js
  • Services: src/services/recipes.js
  • Styling: src/index.css
  • Hosting: firebase.json, .firebaserc

Wichtige Befehle

Lokale Entwicklung:

npm install
npm run dev

Build & Deployment:

npm run build
# interactive login on your machine
npx firebase login
npx firebase deploy --only hosting:kitch-n-break-34666

Hinweise zu Firebase

Die App wurde mit Ihrem Firebase-Projekt konfiguriert. Zum deployen benötigen Sie eine gültige CLI-Authentifizierung (lokal: firebase login oder CI-Token mit firebase login:ci).
